# Nimake syntax - `Reserve words`

## Reverse words

-   set
-   let
-   ovrt
-   allow
-   onoff
-   add
-   noecho
-   opt
-   addOpt
-   optforfile
-   nimcmpr
-   rufofi
-   rufofis
-   afteropt
-   oneminus

## Set keys

-   mainfile
-   cmdbefore
-   cmdafter
-   multiplecmpl
-   mainfilelist

## Specila chars

-   `"` : For string delimiters
-   `,` : Seperate
-   `#` : Comment

---

## Notes

1. You need to write the words exactly as above and no case ignore

2. Nimake does not support function, if else. Only basic statements and comment
